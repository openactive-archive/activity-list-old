A Physical Activity is an exercise, sport or other form of bodily movement that involves physical effort. The OpenActive Activity List provides a standard, openly licensed list of physical activities that can be used to categorise and describe opportunities for people to be physically active. The list is intended to be used by publishers who are sharing open data according to the schema described in the [[Modelling-Opportunity-Data]] specification.

This document provides:

  • both human and machine-readable views of the activity list
  • a summary of the requirements and editorial process that have informed the structure of the list
  • a description of the process by which the list will be revised and amended

This document is an output of the Openactive Community Group. The community group was established with the objective of facilitating the sharing and use of physical activity data.

The list is published under the Creative Commons Attribution (CC-BY) 4.0 licence.

Introduction

A Physical Activity is an exercise, sport or other form of bodily movement that involves physical effort. Physical activities include not just sports, but also a variety of forms of exercise and fitness classes. An Activity List defines a list of physical activities.

A standardised activity list, that provides unique identifiers, labels and descriptions for physical activities can:

  • support integration of data published by multiple activity providers
  • improve user experience across applications through use of a standard set of activity names and definitions
  • enable better discovery and recommendation tools to enable participants to find more opportunities to be active

These requirements have been the primary motivation for the creation of the OpenActive Activity List and have informed its development. The activity list is specifically intended to support data publishers who are publishing open data according to the [[Modelling-Opportunity-Data]] specification, which defines how to relate events to physical activities.

Following the recommendations of the [[Modelling-Opportunity-Data]] specification, this activity list is defined as a controlled vocabulary using the [[SKOS]] standard.

Editorial Notes

This vocabulary is managed by the [Openactive Community Group](http://www.w3.org/community/openactive/). It is expected that this list will undergo revisions based on the needs of the OpenActive community.

A set of editorial guidelines have been used to help inform development and management of the list.

If you wish to make comments regarding this document, please send them to public-openactive@w3.org (subscribe, archives). You may also submit suggested changes to the github repository.
